# 🛒 E-Commerce Web Application (MERN Stack)

A full-featured e-commerce web application built using the **MERN stack (MongoDB, Express.js, React.js, and Node.js)**. This project provides a seamless online shopping experience with a modern UI and robust backend functionality.

---

## 🚀 Features

- User authentication & authorization (JWT-based)
- Product listing with search and filtering
- Add to cart and wishlist functionality
- Secure checkout process
- Order management system
- Admin dashboard for managing products, users, and orders
- Responsive design for mobile and desktop

---

## 🛠️ Tech Stack

**Frontend:**
- React.js  
- Redux Toolkit / Context API  
- Axios  
- Tailwind CSS / CSS  

**Backend:**
- Node.js  
- Express.js  

**Database:**
- MongoDB (Mongoose)  

**Authentication:**
- JWT (JSON Web Tokens)  

**Other Tools:**
- REST APIs  
- Cloud storage (optional for images)
